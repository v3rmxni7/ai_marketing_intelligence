# AI Marketing Intelligence

An agentic AI-based marketing intelligence system that analyzes marketing data and generates actionable insights using modular agents and LLM-powered reasoning pipelines.

---

## Project Structure

ai_marketing_intelligence/
├── data/                  # Input datasets
├── src/
│   ├── agents/            # Task-specific AI agents
│   ├── api/               # API layer
│   ├── domains/           # Marketing domain logic
│   ├── llm/               # LLM wrappers & prompts
│   ├── pipeline.py        # Orchestration pipeline
│   └── utils.py           # Utility functions
├── app.py                 # Application entry point
├── requirements.txt
└── README.md

---

## Overview

- Agentic architecture with autonomous AI agents  
- Domain-driven marketing intelligence  
- LLM-powered reasoning pipeline  
- Scalable and production-ready structure

---

## Setup & Run

pip install -r requirements.txt  
python app.py
